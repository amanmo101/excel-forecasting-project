# Event Revenue & Expense Forecasting (Excel)

An Excel-based forecasting model that projects revenue and expenses across multiple event categories — **weddings, birthdays, and business invitations** — over a several-month period. The workbook combines budget forecasting, trend and distribution visualization, goal-seeking with Solver, and a pivot table for dynamic summarization.

## 📊 Project Overview

This project simulates how a small event/invitation business might plan and track its finances. It forecasts monthly revenue and associated expenses for each event category, then uses Excel's analytical tools to visualize trends, identify revenue distribution patterns, and determine what changes are needed to hit specific monthly revenue goals.

**🗂️ What's Included**

| Component | Description |
|---|---|
| **Budget Forecast** | Month-by-month projected revenue and expenses for each event category (weddings, birthdays, business invitations). |
| **Trend Chart** | Line/column chart visualizing how revenue and expenses move over the forecast period for each category. |
| **Distribution Graph** | Visualization showing how total revenue is distributed across the three event categories. |
| **Solver Analysis** | Uses Excel Solver to calculate the required increase (e.g., in units sold, price, or category mix) needed to reach a target revenue goal for a specific month. |
| **Pivot Table** | Dynamic summary table for slicing revenue and expenses by category, month, or other variables. |

## 🛠️ Tools & Features Used

- Excel formulas for forecasting (e.g., growth assumptions, seasonal adjustments)
- Native Excel charts (line/trend and pie/bar distribution)
- **Solver Add-in** for goal-seeking optimization
- **PivotTable** for interactive data summarization
